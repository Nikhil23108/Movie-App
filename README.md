🎬 Movie App
A responsive Movie Application built using React.js that allows users to explore popular and top-rated movies, search movies by name, and view ratings using data fetched from an external Movie API.

🚀 Features
🔍 Search movies by title
🎥 Browse Popular Movies
⭐ View Top Rated Movies
🏠 Home Page with movie listings
📱 Responsive UI
🖼️ Movie posters and ratings display
⚡ Fast and interactive user experience
🔄 Dynamic data fetching using Axios
🛠️ Tech Stack
React.js
React Router DOM
Axios
CSS
JavaScript
Movie API (TMDB API)

📂 Project Structure
src/ ├── components/ ├── pages/ ├── App.js ├── index.js ├── styles/ └── assets/
